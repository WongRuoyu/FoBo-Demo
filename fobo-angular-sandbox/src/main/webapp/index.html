Although this demo uses the lift-advanced-bs3 Template Application with FoBo's Bootstrap v3.x setting as it's base the following sections is mostly about Angular JS and Lift 3's Round Trip concept. To make the examples a bit more realistic/exciting we are also using Slick to handle the back end storage. The examples is increasingly complex in what they accomplish but considering the "normal complexity" involved in coding the same thing using other techniques the code needed using Lift and Angular is very elegant and simple to comprehend.

I have tried to keep the amount of code used as simple as possible to make the examples clear, for that reason the Slick integration is close to the bare minimum (using a in memory db) and dose, for that reason, not reflect what you would expect in a production ready project.
